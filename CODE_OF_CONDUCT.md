# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

## Summary

We pledge to make participation in this project a welcoming experience for everyone, regardless of background or identity.

## Expected Behavior

- Be respectful and considerate.
- Welcome constructive feedback.
- Focus on what is best for the community.
- Show empathy toward other contributors.

## Unacceptable Behavior

- Personal attacks or insults.
- Trolling or deliberately disruptive comments.
- Publishing others' private information without permission.
- Conduct that would be inappropriate in a professional setting.

## Enforcement

Report concerns by opening a confidential issue or contacting the maintainers listed in [CONTRIBUTING.md](CONTRIBUTING.md). Maintainers may remove comments, commits, or contributions that violate this code.
